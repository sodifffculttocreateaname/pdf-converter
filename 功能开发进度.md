## 四、依赖要求

### 核心依赖（所有功能必需）
- `pypdf` - PDF处理基础库
- `Pillow` - 图片处理
- `PyMuPDF` (fitz) - PDF高级处理

### 可选依赖（特定功能需要）
| 依赖 | 功能 |
|------|------|
| `pdf2image` | PDF转图片、PDF转长图 |
| `pdf2docx` | PDF转Word |
| `pdfplumber` + `openpyxl` | PDF转Excel |
| `python-docx` + `reportlab` | Word转PDF（非COM方式） |
| `openpyxl` + `reportlab` | Excel转PDF（非COM方式） |
| `python-pptx` + `reportlab` | PPT转PDF（非COM方式） |
| `pywin32` + `pythoncom` | Office COM转换（Windows） |
| `pikepdf` | PDF压缩、加密 |
| `chardet` | TXT编码检测 |
